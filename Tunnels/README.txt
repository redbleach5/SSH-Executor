         SSH TUNNEL MANAGER

ЗАПУСК:
   Дважды кликните на START.bat

ТРЕБОВАНИЯ:
   • Python  → python.org/downloads
   • PuTTY   → putty.org
   • Pageant с загруженным SSH ключом

ИСПОЛЬЗОВАНИЕ:
   1. Запустите Pageant, загрузите ключ
   2. Запустите START.bat
   3. Нажмите "Подключить"
   4. Откройте браузер: http://localhost:8080
